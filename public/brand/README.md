# Local Lead Funnel brand assets

This folder contains the editable SVG originals and matching PNG exports for the Local Lead Funnel brand system.

## Visual system

- Navy: `#0F2747`
- Brand blue: `#2563EB`
- CTA green: `#22C55E`
- Light background: `#F8FAFC`
- Text: `#0F172A`
- White: `#FFFFFF`

The source generator is `scripts/generate-brand-assets.mjs`. Run it with the repository's bundled Node.js runtime to regenerate every SVG and PNG after editing the copy or layout.

## Exports

- `logo-square.svg` / `.png`: shared profile image
- `facebook-cover.svg` / `.png`: Facebook cover
- `social-og.svg` / `.png`: website and social share preview
- `post-01.svg` through `post-09.svg` and `.png`: Facebook / Instagram post set
- `highlight-*.svg` and `.png`: five highlight covers
- `stories/story-*.svg` and `.png`: two story cards for each highlight category

The website uses `social-og.png` for Open Graph and Twitter previews, and the footer links to the confirmed brand Instagram account.
